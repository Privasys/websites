## Rust Ecosystem

| Crate | Role in Privasys | License |
|-------|-----------------|---------|
| [rustls](https://github.com/rustls/rustls) | Pure-Rust TLS 1.3 stack used by both the RA-TLS server and HTTPS egress client | Apache-2.0 / MIT |
| [ring](https://github.com/briansmith/ring) | Cryptographic primitives (AES-GCM, ECDSA, HKDF) inside SGX | ISC |
| [rcgen](https://github.com/rustls/rcgen) | X.509 certificate generation for RA-TLS leaf and CA certificates | Apache-2.0 / MIT |
| [webpki](https://github.com/rustls/webpki) | Certificate verification (rustls-webpki) | ISC |
| [Wasmtime](https://github.com/bytecodealliance/wasmtime) | WebAssembly runtime (Privasys SGX fork) for in-enclave WASM execution | Apache-2.0 |
| [serde](https://github.com/serde-rs/serde) | Serialisation framework for configuration and RPC payloads | Apache-2.0 / MIT |
| [x509-parser](https://github.com/rusticata/x509-parser) | X.509 certificate parsing for RA-TLS client verification | Apache-2.0 / MIT |
| [getrandom](https://github.com/rust-random/getrandom) | Entropy source — uses the `rdrand` feature for hardware random inside SGX | Apache-2.0 / MIT |
